Stock Portfolio Tracker

A Python-based Stock Portfolio Tracker that calculates the total value of a user's investments based on the number of shares they own.

Features:
• Stores stock names and their current prices using Python dictionaries.
• Allows users to enter the number of shares they own for each stock.
• Calculates the value of each individual stock holding.
• Displays a complete portfolio summary.
• Calculates the total investment value.
• Generates and saves a portfolio report as a text file.

Concepts Used:
• Python dictionaries.
• Loops and conditional logic.
• User input handling.
• Mathematical calculations.
• File handling (writing reports to a file).

This project demonstrates how Python can be used to create simple financial tracking tools.

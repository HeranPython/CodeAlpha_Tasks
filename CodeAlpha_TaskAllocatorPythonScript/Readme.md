Website Title Extractor

A Python-based web scraping tool that retrieves a webpage and extracts its title automatically.

Features:
• Sends a request to a website using the requests library.
• Retrieves the webpage HTML content.
• Searches for the page title using regular expressions.
• Displays the extracted title.
• Saves the title into a text file for later use.

Concepts Used:
• Python requests library.
• Regular expressions (re).
• HTTP requests and HTML handling.
• File handling.
• Data extraction.

# Website Title Extractor

A Python-based web scraping tool that retrieves a webpage and extracts its title automatically.

## Features
- Sends a request to a website using the `requests` library.
- Retrieves the webpage HTML content.
- Searches for the page title using regular expressions.
- Displays the extracted title.
- Saves the extracted title into a text file for later use.

## Concepts Used
- Python `requests` library
- Regular expressions (`re`)
- HTTP requests and HTML handling
- File handling
- Data extraction

## How to Run
1. Make sure Python is installed on your computer.
2. Install the required library:
   ```bash
   pip install requests
   ```
3. Run the program using:
   ```bash
   python website_title_extractor.py
   ```

## Project Overview
This project demonstrates how Python can interact with websites, extract useful information from HTML content, and automate simple web data extraction tasks.

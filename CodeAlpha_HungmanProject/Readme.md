Hangman Game

A Python-based word guessing game where players try to discover a hidden word by guessing one letter at a time before running out of lives.

Features:
• Randomly selects a word from a predefined word list.
• Allows users to guess letters interactively.
• Tracks guessed letters to prevent repeated guesses.
• Displays the progress of the hidden word after each guess.
• Provides a limited number of lives and shows whether the player won or lost.
• Allows users to play multiple rounds without restarting the program.
• Validates the player's choice when deciding whether to play again.

Concepts Used:
• Python random module.
• Lists and variables.
• Loops and conditional statements.
• User input handling and validation.
• String manipulation.

How to Run:
• Make sure Python is installed on your computer.
• Download or clone this repository.
• Run the program using:

python hangman.py

Project Overview:
This project demonstrates how Python can be used to create an interactive console-based game by combining random selection, loops, conditional statements, and user input processing.
